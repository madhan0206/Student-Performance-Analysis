# Student Performance Analysis System

Python-based data analysis project using a real-world Kaggle student performance dataset.

## Technologies Used
- Python
- Pandas
- Matplotlib

## Features
- Data cleaning and preprocessing
- Total and average score calculation
- Top performer identification
- Subject-wise and gender-wise analysis
- Data visualization using charts
